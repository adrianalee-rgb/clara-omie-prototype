How to connect

1. Server URL

https://api.clara.team/mcp

2. Authentication header

clara-mcp-token: YOUR_TOKEN_HERE

3. Claude / supergateway config

{ "mcpServers": { "clara": { "command": "npx", "args": [ "-y", "supergateway", "--sse", "https://api.clara.team/mcp" ], "env": { "clara-mcp-token": "YOUR_TOKEN" } } } }
